⚙️ Environment Setup
1️⃣ Install Dependencies
npm install express mongoose dotenv
2️⃣ Create Environment Variables

Inside the .env file:
MONGO_URI=your_mongo_connection_string
PORT=3000

🚦 Running the Backend

Use the following command:
node index.js

The server will start at:
👉 http://localhost:3000
